A College Voice Assistant is an AI-powered web application designed to help students, faculty, and staff by providing instant voice-based responses to common college-related queries such as timetables, notices, events, departments, faculty information, and more.
This project uses Artificial Intelligence, Natural Language Processing (NLP), and Speech Recognition to make college information easily accessible through voice commands.
